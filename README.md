# FRAM Website - PRO1001 Frontend Essentials

## Prosjektbeskrivelse

FRAM Website er et frontendprosjekt utviklet som en del av PRO1001 Frontend Essentials. Prosjektet er basert på et utdelt design i Figma og presenterer en bærekraftig matleveringstjeneste som kobler kunder med ferske produkter fra lokale gårder.

Nettsiden består av en forside, en produktside, en kontaktside og en chat-side. Målet var å lage en responsiv og brukervennlig løsning med HTML, CSS og JavaScript, samt å inkludere minst én ekstern API-integrasjon.

## Oppsett og installasjon

Det kreves ingen spesiell installasjon for å vise nettsiden.

Chat-funksjonen krever en API-nøkkel. Av sikkerhetsgrunner er den ekte API-nøkkelen ikke inkludert i repositoryet. For å teste chaten lokalt må du legge inn din egen API-nøkkel i riktig JavaScript-fil der det står placeholder.
